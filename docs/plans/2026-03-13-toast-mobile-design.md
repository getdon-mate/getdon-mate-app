# 모바일 Toast 카드형 디자인

**목표:** 모바일에서 toast가 전체 화면 너비를 차지하지 않도록 줄이고, 하단 중앙에 떠 있는 카드형 피드백으로 개선한다.

## 배경
- 현재 toast는 전체 폭 하단 배너처럼 보여 화면을 무겁게 만든다.
- 참고 이미지처럼 작은 카드형 피드백이 더 가볍고 상태 구분도 분명하다.

## 확정된 방향
- 위치는 하단 중앙 플로팅 유지
- 전체 폭 사용 금지, 적당한 `maxWidth`와 좌우 여백 유지
- 연한 배경 + 진한 텍스트/아이콘 조합
- 좌측 원형 아이콘 배지, 우측 텍스트 블록
- info/success/warning/danger tone 별 배경/테두리/텍스트 색 분리

## 스타일 원칙
- `info`: 연한 블루
- `success`: 연한 그린
- `warning`: 연한 오렌지
- `danger`: 연한 레드
- 그림자, 얇은 border, 큰 radius로 카드 느낌 유지
- 제목과 메시지는 좌측 정렬, 메시지 없으면 더 조밀하게

## 검증 기준
- toast가 화면 전체 폭을 차지하지 않는다.
- 모바일 기준 하단 중앙 카드형으로 보인다.
- tone 별 배경/텍스트가 달라진다.
