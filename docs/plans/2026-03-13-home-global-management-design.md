# 홈 전역 관리 분리 Design

**목표:** 홈 목록 화면에서 앱 전체 관리 진입을 제공하고, 모임통장 상세의 설정 탭은 선택된 모임 내부 관리만 담당하도록 분리한다.

## 배경
- 현재 `AccountListScreen`에는 데모 초기화 같은 테스트성 액션이 남아 있고, 앱 전체 설정 접근은 모임통장 상세의 `SettingsTab`에 묶여 있다.
- 이 구조는 "앱 전체 설정"과 "선택된 모임통장 관리"를 섞어서 탐색을 어렵게 만든다.

## 확정된 방향
- 홈 목록 우측 상단에 전역 아이콘 액션을 둔다.
- 아이콘은 `알림`, `마이페이지`, `앱 설정` 3개를 제공한다.
- `SettingsTab`은 모임통장 기본정보, 자동이체, 1회성 회비, 모임통장 삭제만 남긴다.
- 로그아웃, 회원 탈퇴, 알림 설정, 프로필 관리는 새 `AppSettingsScreen`으로 이동한다.
- `데모 데이터 초기화`는 목록 화면에서 제거한다.

## 화면 책임
- `AccountListScreen`
  - 모임통장 목록 탐색
  - 새 모임통장 생성 진입
  - 전역 화면 진입(`NotificationList`, `MyPage`, `AppSettings`)
- `MyPageScreen`
  - 본인 정보 조회 및 수정
- `AppSettingsScreen`
  - 알림 설정 진입
  - 로그아웃
  - 회원 탈퇴
- `SettingsTab`
  - 현재 선택된 모임통장 내부 관리

## UX 원칙
- 앱 전체 관리와 모임 내부 관리는 같은 화면에 섞지 않는다.
- 홈에서 자주 찾는 전역 관리 진입은 한 번의 탭으로 도달하게 한다.
- 위험 액션은 전역 설정 화면 하단 danger section에 둔다.

## 검증 기준
- 홈 목록 화면에서 데모 초기화 버튼이 보이지 않는다.
- 홈 목록 상단에서 마이페이지와 앱 설정으로 바로 이동할 수 있다.
- 모임통장 상세 설정 탭에 전역 설정 메뉴가 남아 있지 않다.
